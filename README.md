# 🧮 Premium Scientific Calculator

A modern, responsive, and feature-rich **Scientific Calculator** developed using **HTML5, CSS3, and Vanilla JavaScript** as part of the **Oasis Infobyte Web Development & Designing Internship (OIBSIP)**.

The application provides both basic and scientific calculations with a beautiful glassmorphism interface, keyboard support, calculation history, dark/light mode, and local storage.
